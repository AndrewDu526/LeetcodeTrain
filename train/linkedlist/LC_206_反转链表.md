# LC 206 反转链表

## 2. Solution

双指针，构造虚拟节点。两步走：先翻转一对前后节点；再都向后移动一位。注意翻转前需要保存下一个节点；返回的应该是前节点。

## 3. Code Implementation

    class Solution {
        public ListNode reverseList(ListNode head) {
            ListNode pre = null;
            ListNode cur = head;
            ListNode temp = null;
    
            while(cur!=null){
                temp = cur.next;
                cur.next = pre;
    
                pre = cur;
                cur = temp;
            }
            return pre;
        }
    }
