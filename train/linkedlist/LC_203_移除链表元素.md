## 2. Solution

使用虚拟头节点

## 3. Code Implementation

    class Solution {
        public ListNode removeElements(ListNode head, int val) {
            ListNode dummy = new ListNode();
            dummy.next = head;
            ListNode cur = dummy;
            while(cur.next!=null){
                if(cur.next.val == val){
                    cur.next = cur.next.next;
                }else{
                    cur = cur.next;
                }
            }
            return dummy.next;
        }
    }

## 4. Notes

- 注意返回的是dummy.next，避免头节点head被删除
- 用cur.next = cur.next.next来删除节点
