# LC 27 移除元素

## 1. Basic Information

- __level__: easy
- __time-cost__: n/a
- __completed-date__: 1-15-2026
- __completed-times__: 1
- __status__: 不会

## 2. Solution

使用快慢双指针，遍历中每次都将快指针所指元素覆盖慢指针；快指针指向目标元素后慢指针不动，快指针不覆盖。

## 3. Code Implementation

    class Solution {
        public int removeElement(int[] nums, int val) {
            // 快慢指针
            int slowIndex = 0;
            for (int fastIndex = 0; fastIndex < nums.length; fastIndex++) {
                if (nums[fastIndex] != val) {
                    nums[slowIndex] = nums[fastIndex];
                    slowIndex++;
                }
            }
            return slowIndex;
        }
    }
