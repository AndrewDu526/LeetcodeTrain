# LC 108 将有序数组转换为二叉搜索树

## 1. Basic Information

- __level__: easy
- __time-cost__: 超时
- __completed-date__: 1-14-2026
- __completed-times__: 1
- __status__: 回答错误，纠错 

## 2. Solution

取中间元素，左右区间递归。
对于偶数长度的数组，两个元素都可以作为中间值。
正常递归就能满足题干中的“平衡”。

## 3. Code Implementation

    class Solution {
        public TreeNode sortedArrayToBST(int[] nums) {
            int right = nums.length-1;
            return build(nums, 0, right);
        }
    
        private TreeNode build(int[] nums, int left, int right){
            if(left > right) return null;
            int mid = (right + left)/2;
            TreeNode node = new TreeNode(nums[mid]);
            node.left = build(nums, left, mid-1);
            node.right = build(nums, mid+1, right);
            return node;
        }
    }

## 4. Notes

- 纠错：因为right = numms.length-1，设置位闭区间，base case是left > right 而非 ==。
- 纠错：递归时需要更新上下限，缩小左右区间，mid + 1，mid - 1.
