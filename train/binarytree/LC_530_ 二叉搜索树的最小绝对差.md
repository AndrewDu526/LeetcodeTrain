## 2. Solution

双指针，一个指针指向上一个节点。


## 3. Code Implementation

    class Solution {
        int result = Integer.MAX_VALUE;
        TreeNode pre = null;
    
        public int getMinimumDifference(TreeNode root) {
            check(root);
            return result;
        }
    
        private void check(TreeNode node){
            if(node==null) return;
            check(node.left);
            if(pre != null) result = Math.min(result, node.val - pre.val);
            pre = node;
            check(node.right);
        }
    }
