## 2. Solution

用中序遍历后的数组去思考，发现是反中序遍历即右中左，维护一个全局累加变量，对中间node进行累加后赋值。

## 3. Code Implementation

    class Solution {
        int sum = 0;
        
        public TreeNode convertBST(TreeNode root) {
            convert(root);
            return root;
        }
    
        private void convert(TreeNode node){
            if (node == null) return;
            convert(node.right);
            sum+=node.val;
            node.val = sum;
            convert(node.left);
        }
    }

## 4. Notes

- 我一开始想转换成数组后操作数组再转化为二叉树。实际上可直接操作树。
- 由于是从右至左的累加（数组的视角里），因此是反中序遍历。
