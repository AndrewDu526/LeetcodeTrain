## 2. Solution

根据所在节点的大小选择递归左右子树

## 3. Code Implementation

    class Solution {
        public TreeNode searchBST(TreeNode root, int val) {
            if (root == null || root.val == val) return root;
            return root.val > val ? searchBST(root.left, val) : searchBST(root.right, val);
        }
    }
