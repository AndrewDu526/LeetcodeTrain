## 2. Solution

动态规划
1. 定义dp数组：达到第i层的最小花费为dp[i]
2. 推导dp递归公式：dp[i] = Min(dp[i-1]+cost[i-1], dp[i-2]+cost[i-2])
3. 初始化dp数组：dp[0] = 0, dp[1] = 0
4. 遍历顺序，从第二位起，直到超出数组长度
5. 举例证明

注意遍历的起始控制
注意递归公式和cost的关系

## 3. Code Implementation

    class Solution {
        public int minCostClimbingStairs(int[] cost) {
            int len = cost.length;
            int c = 0;
            int a, b = 0;
            for(int i=2;i<=length;i++){
                c = Math.min(a+cost[i-2], b+cost[i-1]);
                a = b;
                b = c;
            }
    
            return c;
        }
    }
