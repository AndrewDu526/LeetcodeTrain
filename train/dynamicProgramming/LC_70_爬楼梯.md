## 2. Solution

动态规划：
1. 定义dp数组：到达第i层的方法有dp[i]个
2. 推导递归公式：dp[i] = dp[i-1] + dp[i-2]
3. 初始化：dp[1] = 1, dp[2] = 2;
4. 遍历
5. 举例

注意我的答案和标准答案在遍历控制/初始化上的差异

## 3. Code Implementation

class Solution {
    public int climbStairs(int n) {
        int c = 0;
        int a = 1;
        int b = 2;
        for(int i=0;i<n;i++){
            c = a + b;
            a = b;
            b = c;
        }
        return c;
    }
}
